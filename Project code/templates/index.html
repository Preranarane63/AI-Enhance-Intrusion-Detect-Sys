Model Design Overview

Aegis IDS utilizes a trained Random Forest Classifier composed of 100 decision estimators. The engine specializes in detecting high-speed modern exploits by classifying traffic flow metrics derived from the standard CICIDS2017 Dataset.

Because security captures generally exhibit high class-imbalance (99% benign traffic), we leverage SMOTE (Synthetic Minority Over-sampling Technique) during pipeline processing to balance malicious classes prior to training. This ensures equal precision across attacks without model bias.
